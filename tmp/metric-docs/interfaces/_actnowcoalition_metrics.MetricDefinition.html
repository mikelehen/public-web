Interface MetricDefinition

A set of parameters that define a given metric (name, data source, thresholds, etc.)

All parameters are optional and should be omitted if default values are okay.

The definition is specified using pure JSON types so that it could be read from a file or CMS.

Optional thresholds

thresholds?: number[]

Thresholds used for grading the metric. These correspond to the levels specified by levelSet and there should be one fewer threshold than there are levels.

Example:

  // Assuming levels are [low, medium, high]
  thresholds: [10, 20]
  // then <=10 is low, 10.1-20 is medium, and >20 is high.

  // Thresholds can be descending as well.
  thresholds: [20, 10]
  // then >=20 is low, 10-19.9 is medium, and <10 is high.
